# YLTextView
##如何使用？
*第一步
  把YLTextView文件夹拖入你的项目
```Objective-C
#include "UITextView+YLTextView.h"
```
  *第二步
  ```Objective-C
    //实例化你的textView
    //两个属性可分别使用，互不影响
    textView.placeholder = @"我是一个占位符";//占位符
    textView.limitLength = @15;//字数限制

  ```
